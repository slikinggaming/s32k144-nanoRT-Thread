# s32k144-nanoRT-Thread
分享一份RT-Thread nano移植到恩智浦s32k144，S32 Design Studio使用的代码。该代码动态线程关闭。线程直接写到main函数里即可。其他自己测吧，发到网上大家自行学习使用。 SDK可用。在此共享以便大家后续维护
